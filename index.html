Sobre Mim

Especialista em Dados & IA

Atualmente estou no 4º semestre de Análise e Desenvolvimento de Sistemas no Centro Universitário IESB, com foco especializado em Data Science e Inteligência Artificial. Minha expertise inclui a criação de pipelines de dados completos, desde a ingestão e limpeza até a análise preditiva e visualização.

Domino tecnologias como Python para análise de dados com Pandas e NumPy, frameworks de machine learning como Scikit-learn и TensorFlow, e ferramentas de BI como Power BI para criação de dashboards estratégicos. Tenho experiência sólida em bancos de dados relacionais (PostgreSQL) e não-relacionais, além de conhecimento em infraestrutura de dados e deploy de modelos de machine learning em ambientes de produção.
