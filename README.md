## 📖 Sobre

Este repositório contém uma página web estática desenvolvida para exibir meu **portfólio** e **currículo**.  
O design foi feito com **HTML, CSS e JavaScript**, garantindo uma interface **simples, moderna e responsiva**.

---

## 📂 Estrutura do Projeto

- `index.html` — Página principal do portfólio.
- `css/` — Folhas de estilo para design visual e responsividade.
- `js/` — Scripts para interatividade e funcionalidades dinâmicas.
- `img/` — Imagens utilizadas na página.
- `Curriculo Gabriel Gomes Almeida.pdf.pdf` — Versão em PDF do meu currículo.

---

## 🛠 Tecnologias Utilizadas

- **HTML5** — Estruturação da página.

- **JavaScript** — Funcionalidades interativas.
